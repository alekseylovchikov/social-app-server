# Social network

### Used technology:
- React.JS;
- Node.JS;
- MongoDB;
- GraphQL.

### How to:
- git clone https://github.com/alekseylovchikov/social-network-merng.git
- cd social-network-merng
- npm install
- add .env file to root folder
  
### ENV variables
- MONGODB - mongodb url
- SECRET - for creating token
